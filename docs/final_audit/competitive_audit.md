## Gap Analysis

### 1. Feature Gaps
* **Dynamic Conversation Explorer**: Competitors like **Profound** and **AthenaHQ** dynamically extract semantic topics, cluster them using embeddings, and display actual query variations. Our platform's Topic Explorer route is 100% mocked with a static SVG and real estate keywords.
* **Citation Gain Attribution**: **Goodie** and **Peec AI** track individual citations and calculate how much visibility was gained. Our Citations tab is static and cannot list actual client backlinks.
* **Model Analytics**: **Profound** allows side-by-side comparison of actual outputs from SearchGPT, Gemini, and Copilot. Our Model comparison is a static mockup.

### 2. Workflow Gaps
* **Self-Service Setup**: Competitors allow a user to sign up, input their domain, automatically discover their competitors, and choose a tracking tier. Our platform lacks a frontend registration route and requires manual database seeding or developer login backdoors.
* **Agency Client Management**: **AthenaHQ** provides white-label client views. Our platform's agency portal is simply a "Coming Soon" screen.

### 3. Reporting Gaps
* **Out-of-the-Box PDF Generation**: **Profound** and **Goodie** allow users to generate and download branded PDF reports for client pitches. Our platform can only email HTML reports.

### 4. Data Gaps
* **Real-time LLM Scrapes**: Our dev environment uses a mock response list because API keys are missing. This restricts actual analysis.
